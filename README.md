# 🎵 Music Player

A responsive and interactive **Music Player** built using **HTML, CSS, and JavaScript**. This project allows users to play, pause, skip tracks, adjust the volume, and navigate through a playlist with a clean and user-friendly interface.

## 📌 Project Overview

The Music Player is a browser-based application that provides an enjoyable music listening experience. It features essential playback controls, a playlist, album artwork, and a responsive design that works across different screen sizes.

## ✨ Features

- ▶️ Play and Pause songs
- ⏮️ Previous and ⏭️ Next track navigation
- 🎵 Playlist with multiple songs
- 🖼️ Album artwork for each song
- 📊 Interactive progress bar with seek functionality
- ⏱️ Current playback time and total duration display
- 🔊 Volume control
- 📱 Responsive design for desktop and mobile devices
- 🎨 Clean and modern user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
