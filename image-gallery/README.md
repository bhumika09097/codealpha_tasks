# 📸 Image Gallery

An interactive and responsive Image Gallery built using HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Development Internship. The gallery allows users to browse images, filter them by category, apply image effects, and view images in a fullscreen lightbox.

## 🚀 Features

- Responsive image gallery layout
- Category-based image filtering
- Image effects:
  - Original
  - Black & White
  - Sepia
  - Contrast
  - Brightness
  - Blur
  - Vibrant
- Interactive lightbox preview
- Previous/Next image navigation
- Smooth hover animations
- Mobile-friendly design
- Lazy loading for improved performance

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome

## 🎯 Functionalities

### Image Filtering
Users can filter images by category:

- All
- Nature
- City
- Animals
- Travel

### Image Effects
Apply different visual effects to gallery images:

- Original
- B&W
- Sepia
- Contrast
- Bright
- Blur
- Vibrant

### Lightbox Preview
- Click any image to open it in fullscreen mode.
- Navigate through images using Previous and Next buttons.
- Close the lightbox anytime.

## 📱 Responsive Design

The gallery is fully responsive and adapts to:

- Desktop
- Tablet
- Mobile devices

## 🔧 Installation

1. Clone the repository:


2. Navigate to the project folder:

```bash
cd codealpha_tasks/image-gallery
```

3. Open `index.html` in your browser.
