# 🎯 CodeAlpha Frontend Development Internship

Welcome to my **CodeAlpha Frontend Development Internship** repository. This repository contains the tasks I completed during my internship, showcasing my frontend development skills using **HTML, CSS, and JavaScript**.

## 📌 Repository Overview

This repository includes the following projects:

### 🎵 Task 1: Music Player
A responsive music player with an attractive user interface and essential audio controls.

#### Features
- ▶️ Play & Pause songs
- ⏭️ Next & Previous controls
- 📊 Interactive progress bar
- 🔊 Volume control
- 🎼 Dynamic song information
- 📱 Fully responsive design

**Technologies Used**
- HTML5
- CSS3
- JavaScript

---

### 🖼️ Task 2: Image Gallery
A modern and responsive image gallery with category filtering, image effects, and lightbox preview.

#### Features
- 🗂️ Category-based image filtering
- 🎨 Image effects (Original, B&W, Sepia, Contrast, Bright, Blur, Vibrant)
- 🔍 Lightbox image preview
- ⬅️➡️ Previous & Next image navigation
- 📱 Responsive layout
- ✨ Smooth animations and hover effects

**Technologies Used**
- HTML5
- CSS3
- JavaScript
- Font Awesome
